# Odoo DevOps Setup

This project demonstrates a Docker-based Odoo 16 deployment using:

- Nginx (reverse proxy)
- Odoo 16
- PostgreSQL 13
- Docker & Docker Compose

## Features

- HTTP reverse proxy using Nginx
- Odoo accessible via port 80
- Modular containerized architecture

## Tech Stack

- Docker
- Docker Compose
- Nginx
- Odoo 16
- PostgreSQL

## Status

- HTTP: Working
- HTTPS (SSL): Not configured yet
